#INC-0924 Incidência

Falha de conexão VPN - Filial Sul

Alta Em Aberto

description Descrição do Problema

Usuários da filial sul (Porto Alegre) estão reportando incapacidade de conectar à VPN corporativa desde as 08:00 de hoje. O cliente Cisco AnyConnect retorna o erro "Connection attempt has timed out. Please verify Internet connectivity." Já verificamos que o link de internet local está operacional e outros serviços em nuvem estão acessíveis. O problema parece estar restrito ao túnel VPN.

psychology Análise da IA (Claude)

Resumo do diagnóstico preliminar:

Com base na descrição e no log anexado, o erro "Connection attempt has timed out" combinado com o código específico no log sugere uma falha de negociação de certificado de segurança após a última atualização do Windows (KB5031356).

lightbulb

Solução Sugerida:

Reinstale o certificado raiz da VPN no repositório local das máquinas afetadas ou reverta temporariamente o patch KB5031356 enquanto a correção definitiva não é homologada. Consulte o KBA-492 para o script de automação.
